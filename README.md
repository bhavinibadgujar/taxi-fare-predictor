# 🚖 Taxi Fare Prediction System

> AI-powered ride fare prediction simulating Uber & Ola dynamic pricing using Random Forest and XGBoost.

## 📁 Project Structure
```
taxi_fare_prediction/
├── data/
│   └── taxi_fare_data.csv      ← Auto-generated dataset (10,000 rides)
├── models/
│   ├── train_model.py          ← Dataset generation + model training
│   └── fare_model.pkl          ← Saved best model
├── app/
│   └── app.py                  ← Streamlit web application
├── requirements.txt
└── README.md
```

## 🚀 Quick Start (3 Steps)

### Step 1 — Install Dependencies
```bash
pip install -r requirements.txt
```

### Step 2 — Train the Model
```bash
python models/train_model.py
```
This generates the dataset and trains both Random Forest and XGBoost models, saving the best one.

### Step 3 — Launch the App
```bash
streamlit run app/app.py
```
Open your browser at `http://localhost:8501`

## 🧠 How It Works

### Features Used
| Feature | Description |
|---|---|
| `distance_km` | Haversine distance between pickup & drop |
| `hour` | Hour of day (0–23) |
| `day_of_week` | 0=Monday … 6=Sunday |
| `passengers` | Number of passengers (1–6) |
| `weather` | 0=Clear, 1=Rain, 2=Storm |
| `is_peak` | 1 if peak hour, else 0 |
| `surge_multiplier` | Dynamic surge (1.0x – 3.0x) |

### Surge Pricing Logic
```python
surge = 1.0
surge += 0.5   # Morning peak  (8–10 AM)
surge += 0.6   # Evening peak  (5–8 PM)
surge += 0.4   # Weekend night (Sat/Sun 8 PM+)
surge += 0.3   # Rainy weather
surge += 0.7   # Storm
# Capped at 3.0x
```

### Fare Formula
```
Fare = Base(₹30) + Distance × ₹14 × Surge + Passengers × ₹2
```
